# Setup Agent

A macOS menu bar app that listens for a Korean wake word (e.g. "자비스", "지니야") and
launches and arranges your apps, browser, and terminal — hands-free.

## Download

**[⬇︎ Download the latest release](https://github.com/UnripePlum/SetupAgent/releases/latest)**

Requires **macOS 14 (Sonoma) or later**. The app is signed with a Developer ID and
notarized by Apple, so it opens without a Gatekeeper warning. It keeps itself up to date
automatically (Sparkle) — or check manually from the menu bar ▸ *Check for Updates…*

## What it does

- Listens for your chosen Korean wake word and launches a set of apps/windows on cue.
- Places launched windows on the display you want.
- Runs quietly from the menu bar.
